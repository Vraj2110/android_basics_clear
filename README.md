📱 Android Studio Exploration Project

This project is created to explore the fundamentals of Android app development using Android Studio. It focuses on understanding layouts, views, and basic UI components.

🚀 What I’m Learning
Android project structure
XML layouts
Different types of Views
UI design basics
Event handling (clicks, inputs)
🧱 Layouts Covered
LinearLayout – arranging elements vertically or horizontally
RelativeLayout – positioning views relative to each other
ConstraintLayout – flexible and modern UI design
FrameLayout – simple layout for stacking views
🎨 Views Used
TextView – display text
EditText – user input
Button – clickable actions
ImageView – display images
RecyclerView (optional) – list of items
